Bem-vindo ao repositório do **Desafio 01** do grupo de estudos focado em Kubernetes e práticas de SRE/DevOps!

## 🎯 Objetivo do Projeto

A proposta deste grupo é **aprender fazendo**. Nosso foco é aprofundar os estudos em Kubernetes construindo projetos práticos que simulam o dia a dia de SRE e DevOps em um ambiente produtivo real. Queremos entender os componentes internos do cluster e como eles interagem na prática, saindo da teoria e colocando a mão na massa.

## 🚀 O Desafio 01 (Primeira Fase)

Nesta primeira fase, o objetivo é preparar o terreno: desde a criação de uma aplicação simples até a sua implantação em um cluster local utilizando práticas modernas de GitOps com Argo CD.

**Checklist de Atividades:**
- [x] Criar este repositório no GitHub.
- [ ] Criar uma aplicação web leve (ex: Go, Python/Flask, ou Node.js).
- [ ] Escrever o `Dockerfile`, realizar o build da imagem e publicar no Docker Hub.
- [ ] Montar um laboratório local com Kubernetes (utilizando `k3d`, `kind` ou minikube).
- [ ] Criar um *namespace* dedicado para a aplicação (não utilizar o `default`).
- [ ] Fazer o deploy do Argo CD no cluster local.
- [ ] Configurar o HPA (Horizontal Pod Autoscaler) para a aplicação web.
